# PicoEvents

PicoEvents è un [portale di classifiche online](https://results.picoevents.ch) per le gare di corsa d'orientamento, usato per gran parte delle gare nazionali e dal 2026 adottato anche per le gare TMO dell'ASTI.  
  
Questa sezione contiene le seguenti guide d'utilizzo:  

- Per gli organizzatori:  
    - [Creazione gara](creazione_gara.md)   
    Descrive come creare una gara per le classifiche in tempo reale al centro gara.  
    Inoltre, spiega come integrare punti radio per tempi intermedi nelle classifiche online.  
- Per i concorrenti e gli organizzatori: 
    - [Visualizzazione](visualizzazione.md)  
    Descrive come visualizzare le liste di partenza e le classifiche online.  
    Per gli organizzatori è utile vedere i concorrenti mancanti.
